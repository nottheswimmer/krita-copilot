Krita Copilot

Krita Copilot is a Krita plugin for creating, editing, and reimagining images with Dalle.

How to install

Copy the krita_copilot folder and krita_copilot.desktop to your pykrita folder. You can find the pykrita folder by opening Krita and going to Settings > Manage resources... > open resource folder. Then restart Krita and enable the plugin in Settings > Configure Krita > Python Plugin Manager.
